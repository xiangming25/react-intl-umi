# react-intl-umi README

一个提供 `react-intl`，`umi-locale` 的代码提示插件

## Features

支持使用 `umi` + `umi/locale` 创建的项目的国际化提示

## Requirements


## Extension Settings

* `react-intl-umi.configPath`: 当前国际化文本内容的位置
* `react-intl-umi.suffix`: 国际化文件的后缀
* `react-intl-umi.regExp`: 在代码中的匹配规则
